SF Base Election Data (SFBED)

This is a sample web site to illustrate using the data from the SFBED open data project.

The data for this web site comes from a single, static JSON file of structured data about San Francisco elections, which can be seen here: data/sf.json. This file is made available under the Public Domain Dedication and License v1.0, which can be read here: data/LICENSE.txt.

If you notice any issues about the data or have a suggestion, please file an issue in the issue tracker.
